# Impacto de la alimentación en el rendimiento deportivo

## Objetivo
Analizar cómo la dieta y los hábitos alimenticios de los deportistas afectan su rendimiento en entrenamientos y partidos.  
Los resultados ayudarán a entrenadores y nutricionistas a optimizar planes alimenticios personalizados.

## Estructura del repositorio
- `data/`: Contiene los datasets en CSV, Excel o JSON.
- `scripts/`: Código para limpieza, análisis y modelado.
- `notebooks/`: Notebooks con análisis exploratorio y visualizaciones.
- `visualizaciones/`: Gráficos generados durante el análisis.
- `reports/`: Informes en Quarto o PDF.
- `docs/`: Documentación adicional del proyecto.

## Tecnologías
- Python: pandas, matplotlib, seaborn, scikit-learn, json
- R: dplyr, tidyr, ggplot2, caret, pROC
